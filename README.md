# 4MIN Portfolio

Personal portfolio website built with **React + TypeScript + Tailwind CSS v4**.

## Features

- Multilingual (EN / TR / AR)
- AI Assistant powered by Groq (llama-3.3-70b-versatile)
- Dark mode
- Infinite horizontal project scroll
- Fully responsive
- SEO optimized (Schema.org, sitemap, Open Graph)

## Stack

- **Frontend:** React 19, TypeScript, Vite, Tailwind CSS v4, Framer Motion
- **Backend:** Netlify Functions (Groq API proxy)
- **Analytics:** GoatCounter

## Run Locally

```bash
npm install
npm run dev
```

## Deploy

Push to `main` → Netlify auto-deploys.

Set `GROQ_API_KEY` in Netlify environment variables for the AI assistant to work.
